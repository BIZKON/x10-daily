**Клиент:** Игорь Рыбаков, движение Х10 (~30 885 кламперов), газета «Бизнес-Практика».

## Дизайн-система (для воспроизведения новых документов)

- A4, padding 13mm × 16mm
- Шрифты: Manrope 700/800 (заголовки), Inter 400/500/600 (текст), JetBrains Mono (числа)
- Цвета: red `#E63946`, gold `#D4A24C`, steel `#1F2937`, surface `#FAFAF7`, border `#E5E2DA`
- **Решение по блокам**: НЕ использовать градиенты — пользователь явно отверг их из-за читаемости в правой части. Все смысловые блоки-выноски — сплошной `var(--steel)` фон + белый текст + золотые акценты на `<b>`. Это эталон editor-note и (теперь) next-card.
- Карточки: border-radius 6px, pill-метки секций, grid-2/grid-3
- Технология: HTML+CSS → headless Chromium (`/home/claude/.cache/puppeteer/chrome/linux-131.0.6778.204/chrome-linux64/chrome`) → PDF через playwright. Скрипт `/home/claude/convert.py`. Wait for `document.fonts.ready` + 2 sec.
- ASCII-имена выходных файлов — кириллица в названии файла ломает превью в Claude.
