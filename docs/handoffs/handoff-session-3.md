# Handoff — Х10 Daily, сессия 3 (26 мая 2026)

## Контекст проекта

**Клиент:** Игорь Рыбаков, движение Х10 (~30 885 кламперов), газета «Бизнес-Практика».
**Задача:** превращение ежемесячной PDF-газеты сообщества в ежедневное деловое мини-апп-медиа **«Х10 Daily»** (Telegram + MAX + email + ежеквартальная печать как сувенир).
**Главная отстройка:** жёсткий уход от инфобиза после кейса Шабутдинова (Like Центр, 7 лет колонии, 31.10.2025).

### 03 · Tech Roadmap v1.0 — `/mnt/user-data/outputs/X10-Tech-Roadmap-v1.0.pdf`
PDF, **16 страниц A4, 2.1 МБ**. Технический план M0→M12 со стеком 2026 и AI-пайплайном контента.

**Ключевые числа:**
- **18.9M ₽** на разработку год (внутри Base-OpEx 30.5M ₽ из финмодели)
- ФОТ 17.4M ₽ = **92% бюджета** · инфра+AI+tools = 8% · AI = ~300K ₽/год = 1%
- **FTE ramp 2 → 6** инженерных + 0.5 designer. Каждый найм — gate по триггерам метрик, не календарю
- **AI-пайплайн из 6 агентов**: DraftAgent → NumbersAgent → FactCheckAgent → ToV-Agent → BrevityAgent → AudioAgent → **human gate** → publish
- Себестоимость **$0.45 за статью** (~42 ₽). 110 статей/мес Base = $50/мес pipeline = $330/мес со всеми AI-фичами на пике (с RAG «Спроси Х10»)

**Стек 2026** (актуальный pricing май 2026):
- **Frontend & Edge:** Next.js 16 + PPR + Hono v4 на Cloudflare Workers + Yandex CF фолбэк
- **Data:** Neon Postgres (Frankfurt) + Hyperdrive cache + **Zero (Rocicorp) sync engine** для клампских чатов + pgvector
- **AI:** Claude Sonnet 4.6 ($3/$15) ядро, Haiku 4.5 ($1/$5) префильтр, Opus 4.7 ($5/$25) на FactCheck в политических темах + ElevenLabs через свой proxy на Render + KikuAI Masker (152-ФЗ)
- **Клиенты:** Telegram Mini App SDK 7.x + MAX через VK Platform Bridge — общий код 90%
- **Editor:** Tiptap + Yjs (collaborative)

**6 решений до старта M0:** двойной стек хостинга · Neon Frankfurt · юр.лицо X10 Daily LLC · Anthropic ZDR-контракт · Figma→Tailwind через Style Dictionary · Tiptap (не Sanity)

### 04 · Competitive Analysis v1.0 — `/mnt/user-data/outputs/X10-Competitive-Analysis-v1.0.pdf`
PDF, **16 страниц A4, 1.8 МБ**. Матрица отстройки против 6 игроков медиа-поля РФ по 7 параметрам.

**Шесть конкурентов** (актуальные цифры, верифицировано web_search май 2026):
1. **РБК Pro** — 1500 ₽/мес физлицу, 9600 ₽/год юрлицу, 7 тарифов, 1.5M MAU, партнёрство с Т-Банком
2. **Forbes Russia** — журнал 985 ₽/номер, ~10M reach, 256K+ TG, активные рейтинги
3. **Т-Журнал (Т-Ж)** — 26M MAU, 27 место по трафику в РФ, 17 курсов (4 платных), 500K+ учеников
4. **vc.ru** — 10-13M MAU, $1.29/мес Plus, 85% трафика из поиска, мужская аудитория 64%
5. **Эквиум** — 750 резидентов из 30K заявок, ~800K ₽/год (исторически), 10+ отделений в мире
6. **Synergy** — 47 факультетов, 100K+ студентов, MBA 300K-3M ₽, EMBA 1.2-3.5M ₽, госаккредитация

**Семь параметров оценки:** Цена входа · Аудитория · Контент-фокус · Дистрибуция · Монетизация · Community · Риск-профиль.

**Главные выводы:**
- **Прямого аналога нет** — никто не сочетает бесплатный вход + кламп-сообщество + AI-конвейер + Mini App
- **Две реальные угрозы:** Т-Журнал за внимание (26M MAU), Synergy за кошелёк (EMBA = годовой Premium ×24)
- **Эквиум — партнёр, не противник.** Воронка: Х10 Daily (бесплатно) → Premium 1500 ₽/мес → Кламп → Эквиум 800K ₽/год
- **Совпали ценой с РБК Pro** (1500 ₽/мес) — позиционируемся как замена/дополнение с кламп-сетью и AI
- **Битва — за community-эффект, не за контент.** Контент-уровень Т-Ж + community-эффект Эквиума = нерешённая задача в РФ

**Карта поля 2×2** (оси: цена входа × тип отношений) — Х10 Daily занимает пустой квадрант «бесплатно + сообщество», белое пятно рынка.

### 05 · Editorial Migration v1.0 — `/mnt/user-data/outputs/X10-Editorial-Migration-v1.0.pdf`
PDF, **16 страниц A4, 1.8 МБ**. План миграции 8 журналистов газеты в авторов с подпиской по модели Stratechery/Lenny's/Substack.

**Бенчмарки 2026** (web_search верифицировано):
- Substack: 10% revenue + 3% Stripe = ~13% total, 5M paid из 50M активных, $450M gross writer revenue в 2025, 78% новых subs через recommendations
- Stratechery: $12/мес, ~40K paid, ~$5M ARR (Ben Thompson 12 лет в одиночку)
- Lenny's: $20/мес, 1.2M subs, $4M+/год (подкаст приносит больше, чем newsletter)

**Наша модель — 70/30 в пользу автора** (асимметрия: у нас аудитория, у Substack — нет).

**4 архетипа автора** (по 2 человека на архетип = 8 всего):
- **A1 · Эксперт** (Малышева) — лонгриды 800-1200 слов, M3-M6 миграция
- **A2 · Аналитик** (Романов) — data-разборы 3-5/нед, M2-M5 миграция, пилот всей модели
- **A3 · Репортёр** (Романчук) — кейс-стори с обязательной audio-расшифровкой, M1-M4 миграция (самые ранние)
- **A4 · Опыт** (Дахужев) — рефлексии действующего предпринимателя + Disclosure of Interests, M4-M8 миграция

**Юр.конфигурация — «двойной контракт»:** трудовой на 60K ₽/мес (соцпакет) + договор с самозанятым на revenue share 70/30 (НПД 6%). При превышении 2.4M ₽/год — переход на ИП-УСН.

**Revenue к M12 целевое:** 360 платных подписок суммарно по 4 авторам = **MRR 700K ₽**. Авторам 70% (490K ₽), Х10 30% (210K ₽) на инфра+AI+промо. Романов на Trackers Pro 3500 ₽/мес = **3M ₽/год** = втрое больше штатки в РБК.

**Exit playbook:** автор уносит имя + контент-архив, Х10 удерживает подписчиков, non-compete 6 мес. **Это нормальное событие**, не катастрофа.

**4 волны миграции:** M0 переговоры (8 чел., все в штате) → M1-M2 Волна 1 (Романов+Романчук) → M3-M4 Волна 2 (+ Малышева + 2-й Аналитик) → M5-M8 Волна 3 (+ Дахужев + 2-й Репортёр + Premium-tier) → M9-M12 Волна 4 (последние 2 + revenue share активируется для всех 8).

## Что важно держать в голове

### Контекстные риски (не меняются между сессиями)

- **Кейс Шабутдинова — центральная задача позиционирования.** Любая публикация в Х10 Daily проходит тест «можно ли спутать с Like Центром / БМ / марафонами успеха». Это не один из факторов — это рамка всего проекта
- **Авторский голос Рыбакова — главный актив и главный риск.** С Q2 строим 3-5 авторских голосов рядом (см. Editorial Migration v1.0)
- **Газета не умирает** — становится ежеквартальным премиум-сувениром для форумов КОД Х10 и Business Meet Up. Печать как канал — мертва, печать как артефакт — работает
- **Финмодель честно показала разрыв со стратегией:** 80K DAU и 2K paid — стретч-цель, не Base. Base-реальность — 24K DAU и 700 paid, cash дефицит -12.3M ₽ к M12, нужно ~24M общего капитала на год

### Переговорные позиции (для пилота M0)

- **С редколлегией:** «газета закрывается ежемесячно, мы открываем авторские витрины с потолком дохода в 3× штатной зарплаты». Аргумент перехода — найденная выдуманная цитата Баффета у Романчук в выпуске №5 (скандал-таймбомба)
- **С Эквиумом:** не конкурент, а партнёр. Воронка Х10 → Premium → Эквиум. Договариваться о совместных ивентах и эксклюзивных интервью с резидентами
- **С Anthropic:** ZDR-контракт подписать до первого вызова API — иначе input/output логируются 30 дней (нарушение 152-ФЗ)

### Финальный канон ToV (после трёх сессий правок)

- **Smart Brevity Axios на русском:** Tease / Lede / Why it matters / By the numbers / Yes but / What's next. ≤ 300 слов / 25-30 сек чтения
- **Чёрный список ~30 слов** (полный — в ToV v1.0). Главные: «соборное мышление», «архитектор возможностей», «преображать мир», «миллион сердец», «созидательная энергия», «проявленность»
- **Без регалий больше 2 в шапке.** Без выдуманных цитат вообще. Без длинных предложений с тремя придаточными
- **Цифры с источниками. Глаголы сильные. Цитаты с атрибуцией кто/где/когда**
